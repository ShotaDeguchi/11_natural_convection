# 11_natural_convection

Finite difference simulation of natural convection. 

This is an extension of [Cavity_FDM_NumPy3](https://github.com/ShotaDeguchi/02b_lid_driven_cavity_flow_3D) to a coupled problem of incompressible flow and heat transfer under the Boussinesq approximation.

See the author's blog post [here](https://qiita.com/ShotaDeguchi/items/69ec6e19ff7adc6abf43).
